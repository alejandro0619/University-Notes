## Objectivos
Crear un sistema de facturación que cumpla la siguiente funcionalidad:

 - Registrar usuario
 - Crear pedidos
   - Lista de N productos
   - Producto:
     - Nombre
     - Precio
     - Cantidad 
 - Pedidos registrados:
   - Seleccionar cliente.
 - Estadísticas
   - Cantidad de pedidos
   - Total de vendido ($ - bs)
   - Total ventas (Natural - Jurídico)
   - Total ventas (Sexo)
   - Tipo de pedido (Por cliente)
     - Natural Venezolanos
     - Natural Extranjeros
     - Jurídicos
     - Jurídicos gubernamentales
 - Configuraciones
   - Tasa de cambios
 - Factura:
   - Número de factura
   - Productos
   - Datos del cliente
   - Forma de pago (Opcional)
 - Salir
